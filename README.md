## Overview

Portfolio V2 is a complete redesign of my personal website, built to showcase my work as an AI & Backend Engineer.

Rather than functioning as a traditional portfolio, the website is designed as an engineering showcase—highlighting real-world projects, system architecture, technical decisions, and software engineering practices.

The goal is to communicate not only *what* I have built, but also *how* I approach designing intelligent systems.

---

## Features

- Modern responsive interface
- Interactive engineering-focused project showcase
- AI & backend project architecture summaries
- Experience timeline
- Technical stack visualization
- Resume viewer and download
- GitHub integration
